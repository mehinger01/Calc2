3. Build Understanding

Core routine: Name the concept → sketch or organize → choose a method → set up → solve → interpret/check.

What to notice

Look for structure: rate, area, rotation, product, radical pattern, sequence term, or curve form.

When stuck

Do not push algebra blindly. Write the problem type, identify the givens, and make a smaller version with friendlier numbers.

6. Guided Practice Problems

No answers are shown. Use the guidance prompts to build independence instead of answer-hunting.

Problem 1. Determine whether a sequence converges from its formula.
Guidance: Before solving, write what topic this problem is testing and what method you expect to use.
Problem 2. Find terms of a sequence from a recurrence relation.
Guidance: Create a small sketch, table, or decision tree before doing algebra.
Problem 3. Identify whether a series is geometric.
Guidance: Solve as far as you can, then write one sentence explaining the meaning of your setup.
Problem 4. Classify a p-series.
Guidance: Circle the step where a common mistake is most likely.
Problem 5. Apply the divergence test.
Guidance: Write a similar problem with easier numbers and solve that first if stuck.
Problem 6. Choose between comparison and limit comparison.
Guidance: After finishing, rate confidence from 1–4 and write the next review date.
Problem 7. Use the ratio test on a factorial or exponential series.
Guidance: Before solving, write what topic this problem is testing and what method you expect to use.
Problem 8. Determine absolute or conditional convergence for an alternating series.
Guidance: Create a small sketch, table, or decision tree before doing algebra.

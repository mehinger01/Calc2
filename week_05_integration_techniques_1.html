Week 5: Integration Techniques I

Basic approaches, integration by parts, and trigonometric integrals

Mission for the Week

Use this organizer before the week begins, then revisit it after each study session. The goal is not just to “see” the topic, but to recognize the problem type, choose a method, and explain the reasoning.

Integration by PartsLIATEPythagorean IdentityHalf-Angle

Big Ideas

  • Technique choice is the main skill.
  • Integration by parts reverses the product rule.
  • Trig integrals depend heavily on parity: odd/even powers.
  • Identities are tools for rewriting before integrating.

Formula / Tool Bank

Integration by Parts: ∫u dv = uv − ∫v du
LIATE: Log, Inverse trig, Algebraic, Trig, Exponential
Pythagorean Identity: sin²x + cos²x = 1
Half-Angle: sin²x = (1 − cos2x)/2, cos²x = (1 + cos2x)/2

Understanding Checks

  • ☐ Can I choose u using LIATE?
  • ☐ Can I identify dv and integrate it?
  • ☐ Can I decide what to save when trig powers are odd?
  • ☐ Can I rewrite even powers using half-angle formulas?

Common Mistakes to Watch

  • Choosing u poorly.
  • Forgetting to subtract the remaining integral.
  • Not simplifying before integrating.
  • Using trig identities in the wrong direction.

Reteach Loop

If stuck: pause, rewrite the problem type in plain English, identify the target quantity, circle the given information, and choose the method before doing any algebra.

  1. What kind of problem is this?
  2. What is the target: value, area, volume, convergence, slope, or model?
  3. What formula or test fits?
  4. What would a wrong setup look like?
  5. Can I explain the first step out loud?
